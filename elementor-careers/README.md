# Elementor Careers Page Import

These files are ready to paste into **Elementor** HTML widgets. All styles are inline (`style=""` on each element) so no external CSS is required.

## How to Import

### Option 1: Section by section
1. In Elementor, add a **HTML** widget for each section
2. Open the corresponding file (01-hero, 02-why-join, 03-open-positions, 04-cta)
3. Copy the entire content and paste into the HTML widget
4. Place widgets in order: Hero → Why Join → Open Positions → CTA

### Option 2: Full template
1. Add one **HTML** widget
2. Open `careers-full-template.html` and copy all content
3. Paste into the HTML widget

## Requirements

- **Montserrat font** – Add in WordPress (Customizer or theme) or use a plugin
- **Font Awesome** – Ensure icons load (Elementor Pro or a plugin like “Font Awesome 4/5/6 Support”)
- Update the **Get In Touch** link in section 04 to your contact page URL (e.g. `/contact/` or full URL)
- Update **mailto** links in section 03 if your careers email is different
